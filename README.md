# 🐍 Premium Snake Game

A modern multi-level Snake game built using HTML5 Canvas, CSS3, and Vanilla JavaScript featuring responsive gameplay, dynamic level systems, animated UI, obstacle mechanics, mobile controls, and persistent high score tracking.

---

# ✨ Features

## 🎮 Gameplay Features

- Multi-level progression system
- Dynamic game speed increase
- Obstacle-based gameplay mechanics
- Real-time score tracking
- High score saving using LocalStorage
- Smooth snake movement
- Collision detection system
- Responsive controls

---

# 🧠 Game Mechanics

## Level System

### Level 1
- Basic gameplay
- Standard snake movement
- No obstacles

### Level 2
- Increased speed
- Wall obstacle mechanics
- Rounded snake visuals

### Level 3
- Advanced obstacle layout
- Faster gameplay
- Circular snake rendering
- Complex navigation challenges
